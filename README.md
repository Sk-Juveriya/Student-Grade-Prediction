# 🎓 Student Grade Prediction using Machine Learning

## 📌 Project Overview
This project predicts student final grades based on:
- Study Hours
- Attendance
- Assignments

A **Decision Tree Regressor** model is used to learn patterns and predict student performance.

---

## 📊 Dataset
A synthetic dataset is generated using NumPy with:
- 120 student records
- Features: Study Hours, Attendance, Assignments
- Target: Final Grade (0–100)

---

## 🤖 Machine Learning Model
- Algorithm: DecisionTreeRegressor
- Library: Scikit-learn
- max_depth = 5
- random_state = 42

---

## 📈 Evaluation Metrics
- RMSE (Root Mean Squared Error)
- R² Score (Accuracy approximation)

---

## 📊 Visualizations
The project includes:
- Study Hours vs Grade
- Attendance vs Grade
- Assignments vs Grade

(All displayed in a single matplotlib window using subplots)
